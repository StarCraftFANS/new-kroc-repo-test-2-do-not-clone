Module course.cycles - Demonstration cycles

Demonstration cycles.

This is part of the course library. To use this library:

#INCLUDE "course.module"

Index

  • Process delta - Duplicate a stream of numbers
  • Process id - Copy input to output
  • Process plus - Add two streams of numbers together
  • Process prefix - Output a fixed number, then copy input to output
  • Process succ - Add one to input numbers
  • Process tail - Drop the first input number, then copy input to output
